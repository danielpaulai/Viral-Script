# SECTION 2: CRITICAL GAPS (What's Missing)

## 🔴 GAP 1: Reference Script Feature (NOT BUILT)

**User Feedback:**
> "There must be an option of uploading your own reference script. If we liked a particular script of a creator, we can paste their reference script and based on that it generates a script for us."

**What's Needed:**
- [ ] Reference Script input field in UI
- [ ] Script Analysis Engine (extract hook, structure, tone, pacing)
- [ ] Pattern matching to 50 hooks
- [ ] Style replication in generation
- [ ] "Analyze Script" button with visual breakdown

**Deliverable Needed:** Complete Reference Script Analysis Prompt + UI Spec

---

## 🔴 GAP 2: Video Type Selection (NOT BUILT)

**User Feedback:**
> "All the videos any person does is not just Talking Head... Some videos you don't speak any word and you just show things on screen."

**What's Needed:**
- [ ] Video Type dropdown in UI
- [ ] Different output templates per type
- [ ] Shot list generation for B-Roll types
- [ ] Text-on-screen timing formats
- [ ] Avatar/Clone expression cues

**Video Types to Support:**
1. 🎤 Talking Head
2. 🎬 B-Roll with Voiceover
3. 📝 Text-on-Screen (No Voice)
4. 🤖 AI Avatar / Clone Video
5. 🖥️ Screen Recording / Tutorial
6. 🎭 Mixed Format

**Deliverable Needed:** Video Type Output Templates for Each Format

---

## 🔴 GAP 3: Creator Style Integration (NOT CONNECTED)

**Status:** We have 16 creator profiles in the library, but they're NOT integrated into the app's generation system.

**What's Needed:**
- [ ] Creator Style dropdown in UI
- [ ] Style selection affects hook choice, tone, structure
- [ ] Each style has its own system prompt modifier
- [ ] Preview/description when hovering over style
- [ ] "Custom Style" option using reference script

**Deliverable Needed:** Integration spec connecting Creator Library to Generation Engine

---

## 🔴 GAP 4: Quality Scoring System (NOT BUILT)

**Problem:** Users can't see WHY a script is good or bad.

**What's Needed:**
- [ ] Viral Score (0-100) based on hook strength, structure
- [ ] Readability Grade (Hemingway analysis)
- [ ] Word count validation
- [ ] Hook identification display
- [ ] Auto-improvement if score is below threshold

**Deliverable Needed:** Quality Scoring Algorithm + UI Display Spec

---

## 🔴 GAP 5: Shot List / B-Roll Suggestions (PARTIAL)

**Status:** We have shot list prompts for Gemini, but the app doesn't generate shot suggestions with scripts.

**What's Needed:**
- [ ] Auto-generate shot suggestions for B-Roll video types
- [ ] Timing per shot
- [ ] Visual description per line of script
- [ ] Export format for video editors

**Deliverable Needed:** Shot List Generation Logic for Each Video Type

---

## 🔴 GAP 6: Platform-Specific Optimization (NOT BUILT)

**Problem:** TikTok, Instagram Reels, YouTube Shorts, LinkedIn have different optimal formats.

**What's Needed:**

| Platform | Optimal Length | Style Notes | Hashtag Format |
|----------|---------------|-------------|----------------|
| TikTok | 15-60s | Trendy, fast, hooks in 1s | #fyp #viral |
| Instagram Reels | 30-90s | Polished, aesthetic | Fewer hashtags |
| YouTube Shorts | 30-60s | Educational, value-packed | SEO titles |
| LinkedIn | 30-90s | Professional, insight-led | Industry hashtags |

**Deliverable Needed:** Platform Optimization Profiles

---

## 🔴 GAP 7: Content Calendar Generator (ENTERPRISE - NOT BUILT)

**Status:** Mentioned in Enterprise tier but no actual implementation.

**What's Needed:**
- [ ] 7-day, 14-day, 30-day calendar options
- [ ] Topic suggestions based on pillars
- [ ] Funnel stage distribution
- [ ] Hook/structure variety enforcement
- [ ] Export to spreadsheet format

**Deliverable Needed:** Calendar Generation Algorithm + Template

---

## 🔴 GAP 8: Onboarding / Voice Calibration Flow (NOT BUILT)

**Problem:** Pro tier promises "scripts in YOUR brand voice" but there's no calibration process.

**What's Needed:**
- [ ] Voice DNA questionnaire (10-15 questions)
- [ ] Sample content analysis

**Deliverable Needed:** Voice Calibration Flow + Questions

# SECTION 3: WHAT NEEDS IMPROVEMENT

## 🟡 IMPROVEMENT 1: Better Hook Selection Logic

**Current State:** User manually selects hook.
**Problem:** They don't know which hook fits their topic.

**Improvement:**
- AI suggests top 3 hooks based on topic + funnel stage
- Shows why each hook would work
- Allows override if user wants different hook

---

## 🟡 IMPROVEMENT 2: More Example Scripts Per Hook

**Current State:** We have 20 example scripts.
**Need:** At least 3 examples per hook type = 150 example scripts

This gives:
- Better pattern recognition for AI
- More variety in outputs
- Proof that hooks work across niches

---

## 🟡 IMPROVEMENT 3: Niche-Specific Variations

**Current State:** Generic scripts that could apply to any niche.
**Need:** Niche-specific hook variations:

| Niche | Hook Adaptation |
|-------|-----------------|
| SaaS/Tech | "This API is doing something nobody's talking about." |
| Coaching | "I spent 10 years helping clients before I realized..." |
| E-commerce | "This product went from 0 to $1M in 6 months." |
| Real Estate | "I just closed a deal that broke every rule." |
| Fitness | "I trained for 5 years wrong. Here's what I changed." |
| Finance | "Your financial advisor isn't telling you this." |

---

## 🟡 IMPROVEMENT 4: CTA Library

**Current State:** Generic CTAs mentioned but not systematized.
**Need:** CTA database matched to funnel stage:

| Funnel Stage | CTA Type | Examples |
|--------------|----------|----------|
| TOFU | Engagement | "Comment if you've experienced this" |
| MOFU | Value Exchange | "DM me 'GUIDE' for the free template" |
| BOFU | Conversion | "Link in bio to book a call" |

---

## 🟡 IMPROVEMENT 5: Anti-AI Detection

**Current State:** We avoid AI words.
**Need:** Active anti-AI measures:

- [ ] Vary sentence rhythm (not all same length)
- [ ] Include deliberate imperfections (conversational fillers)
- [ ] Add personality markers specific to creator
- [ ] Hemingway grade check before output
- [ ] Alternative phrasings for common patterns

---

## 🟡 IMPROVEMENT 6: Script Variations

**Current State:** One script per generation.
**Need:** Generate 3 variations per request:

- Variation A: Conservative (safe, proven formula)
- Variation B: Creative (pushes boundaries)
- Variation C: Contrarian (challenges norms)

User picks favorite or combines elements.

# SUMMARY

## Current App Grade: C+

**Why:**
- Good foundation (hooks, structures, creator styles)
- Missing key differentiators (reference scripts, video types, quality scoring)
- User feedback confirms outputs are "average" compared to ChatGPT

## Target App Grade: A

**How to get there:**
1. Reference Script feature (learn from any viral script)
2. Video Type selection (not just talking head)
3. Creator Style integration (generate like Hormozi, Nas Daily, etc.)
4. Quality Scoring (show why scripts are good)
5. Platform-specific optimization (TikTok vs LinkedIn)

**The goal:** Users should be able to upload a viral script, select a creator style, choose their video type, and get a script that's CLEARLY better than what ChatGPT would produce.

That's the bar. Let's build it.
