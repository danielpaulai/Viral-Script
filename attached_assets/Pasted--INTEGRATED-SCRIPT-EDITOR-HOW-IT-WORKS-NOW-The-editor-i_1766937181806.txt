# INTEGRATED SCRIPT EDITOR

## HOW IT WORKS NOW

The editor is **built into generation**, not a separate step.

When you generate a script:
1. AI writes the script
2. AI automatically checks readability
3. AI fixes any issues BEFORE showing you
4. You see a clean, Grade 4-6 script
5. Optional: Click "Show Analysis" to see the verification

## THE KEY DIFFERENCE

**OLD SYSTEM:**
Generate → See mess → Click analyze → See 14 issues → Fix one by one → Hope it's better

**NEW SYSTEM:**
Generate → See clean script → Done

(Editor is there if you want it, but you probably won't need it)

## PROMPT ADDITION FOR EDITOR MODE

Add this to the master prompt:

```
## WHEN USER EDITS AND ASKS FOR HELP

If user modifies the script and asks for suggestions:

1. Check for issues:
   - Sentences over 15 words
   - Banned adverbs
   - Passive voice
   - Complex words
   - Grade level over 6

2. If issues found, show:
   - Which line
   - What's wrong
   - How to fix
   - What it preserves

3. If no issues, say:
   "Your script looks great! Grade [X.X]. No issues."

4. Always offer optional improvements:
   - Punchier alternatives
   - Delivery notation suggestions
   - Rhythm improvements

5. NEVER:
   - Change the hook format
   - Remove key facts
   - Change the tone
   - Add new information
```
